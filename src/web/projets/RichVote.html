RICHVOTE

Résumé:

Site web dynamique complet réalisé dans le cadre d’un projet de groupe à l’IUT. Il devait permettre aux utilisateurs de rédiger des questions pour que d’autres rédigent des propositions. Ces propositions seraient ensuite votées par d’autre pour obtenir l’idée qui convienne le mieux.

Langages:

  • HTML
  • CSS
  • PHP
  • SQL

Objectif:

Permettre aux utilisateurs de rédiger des questions pour que d’autres rédigent des propositions. Ces propositions seraient ensuite votées par d’autre pour obtenir l’idée qui convienne le mieux.

Contexte

Réalisé dans le cadre d’un projet de groupe à l’IUT. C’est un projet qui a été à réaliser en groupe de 4 et qui s’est étalé de septembre 2022 à février 2023.

Outils utilisés

  • Trello afin de plannifier le travail
  • Git pour faciliter le travail en équipe
  • PHPstorm pour coder le site
  • DBeaver pour gérer la base de données

Qu'est-ce que ca m'a apporté?

Durant ce projet je me suis majoritairement occupé de la base de donnée ainsi que de la mise en relation avec le site web via PHP en respectant la structure MVC. J’ai pu découvrir la gestion de projet selon les méthodes agiles SCRUM, segmenter le travail en sprint séparer par des échanges avec le client était quelque chose de nouveau pour moi. Pour la base de données j’ai pu expérimenter la sécurisation et l’automatisation des comportements de la base de données à l’aide des procédures/fonctions, triggers et vue matérialisées.
